Feature

Workflow Automation

Each submission is tagged by category and moves through statuses: submitted → in_review → accepted/in_progress → resolved (or rejected). Admins can update status in real time.

Admin Console
Filter by status/category and update workflow state.
Status Legend
  • submitted: citizen created
  • in_review: triaged by department
  • accepted: committed for action
  • in_progress: implementation started
  • resolved: completed
  • rejected: not feasible/out of scope
